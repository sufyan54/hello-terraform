# Hello Terraform + GitHub Actions

A minimal project to learn GitHub Actions with Terraform.

## What this does

- Creates a simple text file using Terraform
- Runs automatically on every push
- Shows you GitHub Actions in action!

## Files

- `.github/workflows/learn.yml` - GitHub Actions workflow
- `main.tf` - Simple Terraform configuration

## Try it yourself

1. Fork this repo
2. Make a change to main.tf
3. Push to GitHub
4. Watch Actions tab!
